# Elite Motors Vista

Elite Motors Vista is a web application for managing and showcasing vehicles for sale.

## Getting started

Make sure you have Node.js and npm installed (you can use tools like `nvm` to manage Node versions).

```sh
# Step 1: Clone the repository using the project's Git URL.
git clone <YOUR_GIT_URL>

# Step 2: Navigate to the project directory.
cd elite-motors-vista-main

# Step 3: Install the necessary dependencies.
npm install

# Step 4: Start the development server.
npm run dev
```

The app will start in development mode; open the URL shown in your terminal (usually `http://localhost:5173`) in your browser.

## Tech stack

This project is built with:

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

## Building for production

```sh
npm run build
```

This will create an optimized production build in the `dist` folder, which you can deploy to any static hosting provider (e.g. Vercel, Netlify, GitHub Pages, etc.).
